Help

This extension will work only for pages that listed in rules below.

Url pattern must start with a scheme (http/https), which can be substituded with a * sign.
Examples: *.stackoverflow.com, *.github.com/search, http://goo.gl/

Selector can be any CSS selector that matches input element which has to be focused. Blank value means that extension has to detect input element by itself.
Examples: #search input, input.query.

More info about extension cofiguration can be found on its homepage.
